Tag 2: 'p'

El elemento 'p' es un elemento de línea. Se usa para escribir texto de párrafo.

Por ejemplo: escribir el código '< p > Hola! < /p > resultará en el texto 'Hola!'

Tag 3: 'h1 ... h6'

Éstos son los seis tipos de encabezados; mientras más pequeño el número (e.g. 'h1') más grande será la letra del encabedazo. Usualmente, 'h1' se usa para el título de la página.

Por ejemplo: < h1 >:

Título!

< h2 >:

Subtítulo!

< h3 >:

Encabezado 3!

< h4 >:

Encabezado 4!

< h5 >:
Encabezado 5!
< h4 >:
Encabezado 6!

Tag 4: 'div'

Su nombre es la abreviación de la palabra 'división'; a partir de lo anterior, ya sabemos que sirve para crear secciones en la página, o agruparlas.

Por ejemplo, el código '< div class = "miDivisión"> < h2 > Este es el encabezado de una división < /h2 > < p > Este es el texto de la división 'miDivisión'. < /p > < /div >'; resultaría así:

Este es el encabezado de una división

Este es el texto de la división 'miDivisión'.

Tag 5: 'span'

El elemento 'span' es un contenedor de línea que se usa para agrupar varios elementos (como los mencionados anteriormente) para estilizar/"diseñar" visualmente la página.

Por ejemplo, vamos a nombrar marcas de carros en un texto, cada una con un color diferente:

Unas marcas son: Ford , Ferrari , y BMW

Para diferenciar colores, este fue el código: '[...] son: < span style = "color: blue" > Ford < /span >,'.

En la parte de 'style' se define un estilo particular para el texto que sigue (font o color), en la parte de 'color' se pone el nombre del color del texto, o su código Hexadecimal; seguido por el texto que se pondrá en el color mencionado

Este es el link de la referencia para la explicación anterior! :)

Tag 6: 'strong'

En Word, existe la opción de poner un texto en negrilla , no? En HTML, esto mismo se conoce como el elemento 'strong'; es uno de línea. El texto que esté dentro de '< strong >' es lo que se pondrá en negrilla.

Por ejemplo, el código 'texto énfasis ' se escribió así:

'texto < strong > énfasis < /strong >'.

Este es el link de la referencia para la explicación anterior! :)

Tag 7: 'i'

En Word también existe la opción de poner un texto en itálica , no? En HTML, esto mismo se conoce como el elemento 'i'; es uno de lína. Se usa para escribir texto en un todo diferente al usual.

Por ejemplo, el código 'normal, y diferente ' se escribió así:

'normal, y < strong > diferente < /strong >'.

Tag 9: 'b'

En HTML, para compensar estilísticamente una palabra o texto, se usa el elemento 'b'. También es otra opción para hacer énfasis en una parte del texto, al igual que el elemento 'strong'.

Por ejemplo, el código 'texto, y texto diferente ' se escribió así:

Tag 23: 'table'

Al igual que en HTML se puede hacer un "cuestionario" con el tag '< form >', también se puede hacer tablas con el tag '< table >'

Dentro de este tag se definen las columnas de ésta (como puntaje o número) como 'th' (encabezado de tabla), al igual que los contenidos de ésta como 'td' (data de la tabla). También, cada el contenido de cada fila se define como 'tr' (table row). Por ejemplo:

Equipo Puntaje
Red Bull 100
Alpine 90

Este sería el código de la tabla anterior:

< table >
< tr >
< th > Equipo < /th >
< th > Puntaje < /th >
< /tr >
< tr >
< td > Red Bull < /td >
< td > Alpine < /td >
< /tr >
< tr >
< td > 100 < /td >
< td > 90 < /td >
< /tr >
< /table >

Tag 24: 'ul'

Este tag se usa cuando se quiere definir una lista con "bullet points", en una página HTML.

La lista en sí se define dentro de este tag, usando el tag '< li >'; el cual define el elemento de la lista (como 'cuaderno').

Por ejemplo, esta es una lista de útiles de una clase:

  • Bolígrafo
  • Laptop
  • Cuaderno

Y este sería el código:
< ul >
< li > Bolígrafo < /li >
< li > Laptop < /li >
< li > Cuaderno < /li >
< /ul >

En este link aprendemos más sobre este tag! :)

Tag 25: 'ol'

Al igual que se puede definir una lista en HTML con el tag 'ul', con el tag 'ol' se puede definir una lista numerada; puede empezar en el '1' o en el número que uno quiera.

Por ejemplo el código de esta lista, que empieza en el núemro '3' sería:

  1. Bolígrafo
  2. Laptop
  3. Cuaderno

Sería:
< ol start = "3" >
< li > Bolígrafo < /li >
< li > Laptop < /li >
< li > Cuaderno < /li >
< /ol >
